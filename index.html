Welcome to the server! =)

Our server is running at http://localhost:5000/ and is connected to the OWL ontology of Lan Nguyen.

The purpose of this application is to allow the user to parse JSON data and then click on a link-button, to send a SPARQL query to the Stardog triple store.
There are two boxes shown below; the difference between these two is that one query makes use of inferences while the other does not.

With Inferencing Send a SPARQL query with inferencing

This query will run against the repository, and will show any inferences that are present. In this example, it shows that ColdChickenUdon belongs to the class ColdDishes.

Click the link button to GET the query to our triple store. Again, we will wrap this call in a server side function {{ url_for('sparql') }}
